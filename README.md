# Just Magic 3

Unity third-person fantasy prototype.

## Unity Version

Open with Unity `6000.4.7f1`.

## Project Layout

- `Assets/` contains scenes, scripts, models, animations, input actions, and render settings.
- `Packages/` contains Unity package dependencies.
- `ProjectSettings/` contains project configuration.

Generated Unity folders such as `Library/`, `Temp/`, `Logs/`, `obj/`, and `GeneratedAssets/` are intentionally ignored.
